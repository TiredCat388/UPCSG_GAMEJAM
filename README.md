# UPCSG_GAMEJAM

## How to start contributing

1. clone the repository
```bash
git clone https://github.com/TiredCat388/UPCSG_GAMEJAM.git
```
3. download **Godot 4.5.1** either on [Steam](https://store.steampowered.com/app/404790/Godot_Engine/) or [Godot Official site](https://godotengine.org/)
4. Launch the Godot application and click Import on the top-left of the Project Manager.
5. Navigate to the folder where you cloned the repository, then click Select Current Folder.

## Branching / Commit Rules

Branch with your last name. Then commit using the format:

```text
[FeatureName][LastName]commitmessage
```

(e.g [PlayerController][Antig] Added basic player movement)
